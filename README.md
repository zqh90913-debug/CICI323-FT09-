#  Machine Learning Regression Models Project

This repository contains the implementation and evaluation of several regression models using Python.  
It includes the dataset, source code, and screenshots showing the experimental results.

---

##  Project Structure

| File | Description |
|------|--------------|
| `data.csv` | The dataset used for model training and evaluation |
| `models_comparison.ipynb` | The Jupyter Notebook source code that runs all models |
| `results.pdf` | Screenshots showing the execution results, charts, and metrics |

---
