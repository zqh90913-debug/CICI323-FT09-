#  Machine Learning Regression Models Project

This repository contains the implementation and evaluation of several regression models using Python.  
It includes the dataset, source code, and screenshots showing the experimental results.

---

##  Project Structure

| File | Description |
|------|--------------|
| `data.csv` | The dataset used for model training and evaluation |
| `models_comparison.ipynb` | The Jupyter Notebook source code that runs all models |
| `results.pdf` | Screenshots showing the execution results, charts, and metrics |

---

## Operation Guide (Using Google Colab)

This project is designed to be run on **Google Colab**, an online platform for executing Jupyter notebooks without local installation.

### Step-by-step Instructions

1. **Download the dataset**
   - In this repository, click on `data.csv`.
   - Click **“Download raw file”** to save it to your computer.

2. **Open Google Colab**
   - Sign in with your Google account if prompted.

3. **Open the notebook**
   - Click **“File → Upload notebook”**.
   - Upload the `.ipynb` source file .

4. **Import the dataset into Colab**
 can upload programmatically using the following code:
     ```python
     from google.colab import files
     uploaded = files.upload()
     ```
   - When prompted, choose the `data.csv` file.

5. **Run the notebook**
   - Click **Runtime → Run all
